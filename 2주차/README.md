# 📘 수식 변환과 미로 찾기 실습 (2-1.c ~ 2-4.c)

이 저장소는 C 언어로 **스택을 활용한 수식 변환 및 계산**, 그리고 **스택 기반 미로 경로 탐색**을 구현하며 핵심 알고리즘을 실습한 예제 모음입니다. 각 파일은 실전 알고리즘 문제 해결 방식에 초점을 맞추어 구성되어 있으며, 특히 **후위/전위 수식 변환과 계산** 로직을 완전하게 구현하고 있습니다.

---

## 📁 파일 구성

### 🔹 `2-1.c` — 한 자리 피연산자의 후위 표기법 변환
- **주요 내용**: 중위식 → 후위식(Postfix) 변환
- **핵심 포인트**:
  - 스택을 이용한 연산자 우선순위 처리 (`isp`, `icp`)
  - 피연산자는 그대로 출력, 연산자는 스택에서 정렬 후 출력
  - 괄호 처리 및 우선순위 비교를 통한 정확한 변환

---

### 🔹 `2-2.c` — 여러 자리 숫자 포함 후위 표기 & 계산
- **주요 내용**:
  - **여러 자리 정수**를 포함한 중위식 → 후위식 변환
  - 후위식 문자열로 출력 → **후위식 계산기**까지 구현
- **기능**:
  - 연속된 숫자 파싱 (예: 123, 45)
  - 변환된 후위식 문자열 파싱 후 계산 수행 (스택 기반)

---

### 🔹 `2-3.c` — 전위 표기법 변환 & 계산기 구현
- **주요 내용**:
  - 중위식 → 전위식(Prefix) 변환
  - 전위식 기반 계산 수행까지 구현
- **핵심 특징**:
  - 문자열을 뒤에서부터 읽으며 전위식 생성
  - 후위식과는 반대로 연산자 우선순위 적용
  - 생성된 전위식 문자열 역순 정렬 후 계산기 구현
- **계산 방식**: 전위식 역방향 탐색 → 숫자 파싱 → 연산자 적용

---

### 🔹 `2-4.c` — 미로 찾기 (스택 기반 DFS)
- **주요 내용**:
  - 교재 [그림 3.8 ~ 3.10]에 해당하는 **스택 기반 미로 탐색 알고리즘**
- **구현 방식**:
  - 방향 배열 `move[8]`로 8방향 탐색 구현
  - 스택을 이용한 경로 저장 (DFS 방식)
  - `mark` 배열로 방문 여부 추적
- **기능**:
  - 시작점 → 출구까지의 경로가 존재하면 출력
  - 경로가 없을 경우 "The maze does not have a path" 출력

## 📚 학습 목표

| 구현 항목         | 실습 내용 요약 |
|------------------|----------------|
| 후위 표기법 변환  | 스택 기반 연산자 우선순위 적용, 괄호 처리 |
| 후위 계산기 구현  | 정수형 피연산자 파싱 + 후위식 계산 |
| 전위 표기법 변환  | 오른쪽부터 탐색, 연산자 처리 후 전위식 생성 |
| 전위 계산기 구현  | 전위식 역탐색 → 스택 기반 계산 |
| 미로 찾기 (DFS)   | 2D 배열 기반 미로 탐색, 스택으로 경로 백트래킹 |
