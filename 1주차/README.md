# 📘 자료구조 실습 (1-1.c ~ 1-3.c)

이 저장소는 C 언어를 이용한 **기본 자료구조 학습 실습**입니다. 스택과 큐, 다항식 연산을 배열과 연결리스트 방식으로 구현하면서 각 구조의 동작 원리와 메모리 구조를 직접 이해하고 구현해보는 데 목적이 있습니다.

---

## 📁 파일 목록

### 🔹 `1-1.c` — 배열을 이용한 스택 / 원형 큐

#### ✅ 배열 기반 스택 구현
- 고정 크기 `MAX_STACK_SIZE(10)`의 배열로 스택 구성
- `push`, `pop` 동작 구현
- 입력한 문자 10개를 스택에 넣은 후, 꺼내서 역순 출력 (LIFO 확인)

#### ✅ 배열 기반 원형 큐 구현
- 고정 크기 `MAX_QUEUE_SIZE(10)`의 원형 구조 큐 구성
- `addq`, `deleteq` 함수로 삽입/삭제 처리
- 문자 9개를 순차적으로 넣고 빼며 FIFO 구조 확인

---

### 🔹 `1-2.c` — 연결리스트로 스택 / 큐 구현

#### ✅ 연결리스트 스택
- `stackPointer`를 사용한 단일 연결리스트 기반 동적 스택
- 노드 할당/해제를 통해 크기 제한 없이 작동
- 입력 문자 3개를 스택에 넣고 pop 하며 상태 출력

#### ✅ 연결리스트 큐
- `front`, `rear` 포인터를 이용한 단일 연결리스트 기반 큐
- `addq`, `deleteq`로 큐 삽입/삭제 구현
- 각 단계에서 현재 큐 상태를 출력하며 구조 동작 시각화

---

### 🔹 `1-3.c` — 다항식 덧셈 (배열 / 연결리스트)

#### ✅ 배열 기반 다항식
- `terms[]` 배열에 계수(coef), 지수(expon) 쌍 저장
- 두 다항식을 입력받아 `padd()`로 합산하여 결과 배열 생성
- 동일 차수 항은 계수를 더하고, 출력 시 형식화하여 표현

#### ✅ 연결리스트 기반 다항식
- `polyNode` 구조체를 활용한 단일 연결리스트
- 각 노드에 다항식 항 저장, `padd()`로 다항식 A + B 구현
- 연결리스트를 따라 결과 다항식을 출력
